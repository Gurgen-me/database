БАЗА ДАННЫХ

БД - место в которых хранятся данные сайта
столбец - поле, строка - запись
СУБД - система управления базой данных
sql - структурированный язык запросов для работы с бд
реляционные бд - базы где между таблицами есть связи

СУБД которую мы будем использовать - MySQL / mariaDB
phpMyAdmin - веб приложение для удобной работы с бд

INT - целое число
VARCHAR -ограиниченная небольшая строка
TEXT - большой ограниценный текст / строка
DATE - время

SQL в PHP

mysqli_connect = подсключение к бд
mysqli_query - выполнение запроса к базе

mysqli_query($link, $res) or die(mysqli_error($link))

mysqli_fetch_assoc - достать первую запись из результатов
mysqli_fetch_all - достать все
